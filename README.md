# Estudo de Rust — guessing_game

Este repositório é um estudo de Rust baseado no livro: https://rust-book.cs.brown.edu/

Este projeto contém uma implementação simples de um jogo de adivinhação (guessing game) usado para praticar conceitos básicos da linguagem Rust.

Conteúdo

- Código-fonte em `src/` (principalmente `src/main.rs`).
- Manifesto do projeto em `Cargo.toml`.

Como executar

1. Tenha o Rust instalado. Veja https://www.rust-lang.org/tools/install
2. No diretório do projeto, rode:

```
cargo run
```

O comando irá compilar e executar o binário do jogo.
